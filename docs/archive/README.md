# Archivio Documentazione

Questa cartella contiene documenti storici o snapshot superseded.

Regole:
- Non usare questi file come fonte primaria dello stato attuale.
- Per stato operativo corrente, usare `docs/implement-index.md`.
- Per blueprint/architettura corrente, usare `docs/blueprint.md` e `docs/adrs/*`.

Contenuti archiviati:
- `improvement-roadmap.md`: roadmap storica con task e snippet riferiti a uno stato precedente.
- `architecture-review.md`: review architetturale point-in-time.
- `copilot-review-followups.md`: follow-up PR storici non bloccanti.
